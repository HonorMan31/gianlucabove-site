# Pacchetto sito – versione 2

Questa cartella contiene una versione più completa del sito, con:
- home aggiornata
- pagina "Chi sono"
- 7 pagine servizi
- sezione blog con 3 articoli esempio
- immagini segnaposto già visibili
- form contatti predisposto per email

## PASSO 1 – Apri il sito in locale
Fai doppio clic su `index.html` per vedere l’anteprima.

## PASSO 2 – Sostituisci i file su GitHub
Nel repository GitHub che hai già creato:
1. clicca **Add file** → **Upload files**
2. apri questa cartella sul tuo computer
3. trascina dentro GitHub **TUTTO il contenuto** della cartella
4. se GitHub chiede di sostituire i file esistenti, conferma
5. clicca **Commit changes**

## PASSO 3 – Controlla il sito online
Dopo l’upload, ricarica l’indirizzo GitHub Pages del sito.

## PASSO 4 – Collega il modulo alla tua email
Il form è pronto ma devi sostituire l’ID del form.

Nei file cerca questa riga:
`https://formspree.io/f/INSERISCI_ID_FORM`

Quando crei il tuo form su Formspree, sostituisci `INSERISCI_ID_FORM` con l’ID vero.

## PASSO 5 – Aggiungi le tue immagini vere
Le immagini segnaposto stanno in `assets/images/`.
Puoi sostituirle con le tue foto mantenendo gli stessi nomi dei file, così non devi cambiare il codice.

Esempi:
- `profilo-avvocato.svg` → puoi sostituirlo con una tua foto rinominata `profilo-avvocato.jpg` ma in questo caso bisogna aggiornare anche il codice
- soluzione più semplice: dimmelo e ti preparo io i file già modificati

## PASSO 6 – Pubblica nuovi articoli
Per aggiungere un articolo:
1. duplica `blog/template-articolo.html`
2. rinomina il file, ad esempio `nuovo-articolo.html`
3. modifica titolo, descrizione e testo
4. aggiungi il link anche in `blog/index.html`

## NOTA IMPORTANTE
Prima di collegare il dominio principale `gianlucabove.com`, conviene:
- controllare bene tutti i testi
- inserire le immagini reali
- attivare il modulo contatti
- verificare privacy e dettagli finali
